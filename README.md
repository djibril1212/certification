## 🎯 Les deux épreuves

### 1. Le dossier écrit (groupe) — *fait*
Pilotage complet du projet de refonte du portail B2B, du cadrage au retour d'expérience.

| Partie | Contenu |
|--------|---------|
| **5.1** Cadrage & gouvernance | Note de cadrage, périmètre, objectifs SMART, gouvernance, RACI |
| **5.2** Planification & chiffrage | WBS, charges, budget (162 k€), planning, risques, qualité |
| **5.3** Méthodologie & coordination | Choix agile/hybride, backlog, DoR/DoD, rituels, Kanban |
| **5.4** KPI & reporting | Tableau de bord, seuils d'alerte, rapport d'avancement |
| **5.5** Résolution d'incidents | Incident SSO/JWT, recherche documentaire, décision |
| **5.6** Montée en compétences | Atelier OAuth2/JWT, RETEX, indicateurs d'impact |

### 2. La soutenance orale (individuelle) — *préparée dans `Soutenance/`*
- **Modalité :** individuelle · **20 minutes** · présentation orale
- **Compétences évaluées :** **C10** (Planifier et conduire) + **C11** (Coordonner les équipes)
- **Notation :** /25 (5 critères × 5 points : 0 = hors sujet → 5 = professionnel)
- **Rôle présenté :** Chef de projet

➡️ Tout le matériel de préparation est dans **[Soutenance/](Soutenance/)** (commencer par son [README](Soutenance/README.md)).
